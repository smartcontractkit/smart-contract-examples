# Self Automating Contracts

This demo contract illustrates how to create a smart contract that can create automated counters using Keepers.

## Getting started

### Prerequisites

This example uses the [Goerli](https://goerli.net/) network.
You will need:
- Testnet ETH
- Testnet Link
